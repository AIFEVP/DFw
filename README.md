# DFw - Demo Framework

This repository contains various demonstration projects and tools for different technologies and frameworks.

## Contents

- **Aspire**: .NET Aspire application demonstrations
- **BE**: Backend services
- **BlazorNg**: Blazor and Angular integration examples
- **PwTests**: Playwright testing examples
- **electronFw**: Electron framework applications
- **gql**: GraphQL server and related tools
- **mcpTools**: Model Context Protocol (MCP) tools and servers
- **nodeAddOn**: Node.js add-on examples
- **utils**: Utility scripts and tools
- **yashmcp**: Additional MCP tools

## Documentation

- [**GitHub Copilot Agents Guide**](COPILOT_AGENTS.md) - Comprehensive guide explaining the differences between Cloud Agent, Chat, and CLI Agent

## GitHub Copilot Integration

This repository includes examples of MCP (Model Context Protocol) servers that can be used with GitHub Copilot Chat in VS Code and Visual Studio. See the [mcpTools](mcpTools/) directory for more information.

## Getting Started

Each subdirectory contains its own README or documentation for specific technologies and tools. Navigate to the relevant directory for more information.
